# Ceremonies

## Daily sync
- What was completed yesterday?
- What will be completed today?
- What risks threaten demo readiness?
- What needs a decision by the Lead?

## Architecture review (2x per week while building)
- Confirm source contracts and canonical entities
- Confirm Fabric item choices and workload responsibilities
- Review Silver / Gold design changes
- Review parameterization and environment strategy

## Demo readiness review
- Validate all required ingestion methods run
- Validate all required transformation methods run
- Validate KPIs render correctly
- Validate fallback demo data exists if live APIs fail

## Retrospective
- What improved demo clarity?
- What reduced setup friction?
- What should become a reusable pattern or template?
