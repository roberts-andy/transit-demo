# Transit Demo Squad

## Mission
Design and implement a reusable Microsoft Fabric demo that proves multiple ingestion and transformation options using public transit, weather, and public-event data.

## Success criteria
- Functional Bronze / Silver / Gold architecture
- At least one working path for each required Fabric ingestion/transformation method
- Demo narrative is understandable to both technical and executive audiences
- Repo is reusable by other field teams
